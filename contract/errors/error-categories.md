# Error Categories

MCP tools should distinguish:

- `invalid_tool_arguments`
- `unsupported_document_kind`
- `missing_base_state`
- `upstream_runtime_failure`
- `upstream_validation_error`
- `upstream_warning`
- `file_access_policy_error`
- `storage_policy_error`
- `transport_error`
- `session_error`
