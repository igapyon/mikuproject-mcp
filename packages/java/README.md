# Java Placeholder

This directory is a placeholder only.

`mikuproject-mcp` currently uses the Node.js / TypeScript implementation as the
MCP server. Java remains a supported runtime artifact path through
`runtime/mikuproject-java/`, but there is no current Java MCP server
implementation target.

Do not add a Java MCP server here unless a concrete future distribution or
runtime constraint justifies maintaining a second MCP server implementation.
